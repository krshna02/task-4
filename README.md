Responsive Website – Media Query Conversion

This project converts a desktop-only webpage into a fully mobile-friendly responsive layout using CSS media queries.

Features

Responsive design using @media (max-width: 768px)

Mobile-first navigation layout

Images scale properly inside containers

Columns stack vertically on small screens

Fixed widths replaced with fluid, percentage-based layouts

Overflow and horizontal scrolling issues removed

Tech Used

HTML5

CSS3

Chrome DevTools (Device Toolbar for testing)

How to Use

Open the project folder in VS Code.

Run the HTML file using Live Server or any browser.

Resize the browser window or use Chrome DevTools → Device Toolbar to test mobile view.

Modify or extend the media queries in style.css as needed.

Goal

Ensure the webpage looks clean, readable, and functional on mobile screens under 768px width.
